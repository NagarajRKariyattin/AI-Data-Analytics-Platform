# 🚀 AI Data Analytics Platform

An end-to-end AI-powered data analytics platform that enables users to upload business data, perform ETL operations, query data using natural language, generate AI-powered business insights, and visualize results through interactive dashboards.

---

## 📌 Project Overview

Organizations often receive raw data in Excel or CSV format. Before it can be analyzed, the data must be cleaned, transformed, and stored in a database.

This project automates that workflow by combining Data Analytics, SQL, and Generative AI.

Users can:

- Upload Excel or CSV files
- Clean and transform data using Python
- Store processed data in PostgreSQL
- Ask questions in natural language
- Convert natural language into SQL using Ollama
- Generate AI-powered business insights
- Visualize KPIs using Power BI
- Export analytical reports to Excel

---

## 🎯 Project Objectives

- Build a real-world ETL pipeline
- Learn database design using PostgreSQL
- Integrate Generative AI with business analytics
- Develop Natural Language to SQL functionality
- Create interactive dashboards
- Generate automated business reports
